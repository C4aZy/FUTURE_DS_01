# E-Commerce Sales Performance Analysis (October 2019)

## Overview
This project analyzes e-commerce sales data for October 2019 to identify revenue trends, category performance, and customer behavior. The objective is to generate clear, actionable business insights using cleaned data and visual analytics.

## Dataset
- Raw sales data in CSV format
- Time period: October 2019
- Includes product categories, revenue, traffic, and conversion metrics

## Key Objectives
- Clean and organize raw sales data
- Analyze monthly sales trends
- Evaluate category-wise revenue and conversion performance
- Understand customer behavior patterns
- Present insights through an interactive dashboard

## Tools Used
- Data cleaning & preprocessing
- **Google Looker Studio** for visualization and dashboarding

## Dashboard Highlights
- Revenue trend over time
- Revenue by product category
- Average conversion rate by category
- Traffic vs conversion analysis (bubble chart where size = revenue)

## Key Insights
- Diapers generate the highest revenue with strong conversion rates
- Sales peak during mid-October, indicating seasonal or promotional impact
- Electronics receive very high traffic but relatively low conversion
- High-traffic, low-conversion products indicate pricing or positioning issues

## Conclusion
This analysis demonstrates how structured data cleaning and visual storytelling in Looker Studio can support better business decisions. The dashboard can be extended for multi-month or year-over-year analysis.
